# FCP TypeScript SDK

Official TypeScript/JavaScript SDK for the [Food Context Protocol](https://github.com/Food-Context-Protocol/fcp).

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)
[![npm](https://img.shields.io/npm/v/@fcp/sdk)](https://www.npmjs.com/package/@fcp/sdk)

## Installation

```bash
# npm
npm install @fcp/sdk

# yarn
yarn add @fcp/sdk

# pnpm
pnpm add @fcp/sdk
```

## Quick Start

```typescript
import { FCPClient } from "@fcp/sdk";

// Initialize client
const client = new FCPClient({
  baseUrl: "https://fcp.dev",
  apiKey: "your-api-key"
});

// Analyze a meal photo
const result = await client.analyze.analyzeMeal({
  imageUrl: "https://example.com/food.jpg"
});
console.log(result.nutrition);

// Search recipes
const recipes = await client.recipes.searchRecipes({
  query: "high protein breakfast",
  maxResults: 5
});

// Check food recalls
const recalls = await client.safety.checkFoodRecalls({
  foodItem: "romaine lettuce"
});
```
